# 摄像头运维数据采集工具 — 手机安装使用说明

## 一、访问地址

**确保手机和电脑连接的是同一个 Wi-Fi 网络。**

在手机浏览器地址栏输入：

```
http://192.168.246.178:8080
```

> 如果电脑 IP 变了，在电脑终端运行 `ipconfig getifaddr en0` 查看新 IP。

---

## 二、添加到手机桌面（PWA安装）

### iPhone（Safari 浏览器）

1. 打开 Safari，输入上方地址
2. 等待页面加载完成
3. 点击底部 **「分享」按钮**（方框加箭头图标）
4. 向下滑动，选择 **「添加到主屏幕」**
5. 点击右上角 **「添加」**
6. 桌面会出现「运维采集」图标，下次直接点它打开

> ⚠️ 如果无法安装，确认 Safari 设置中已开启：
> - 设置 → Safari → 高级 → JavaScript（已开启）
> - 设置 → Safari → 高级 → 实验性功能 → Service Worker（已开启）

### Android（Chrome 浏览器）

1. 打开 Chrome，输入上方地址
2. 等待页面加载完成
3. 地址栏下方或底部会出现 **「安装」弹窗**，点击安装
4. 或点右上角菜单（⋮）→ **「添加到主屏幕」**
5. 桌面会出现「运维采集」图标

## 三、基本操作

### 新增记录

1. 点击底部 **「新增」** 按钮
2. 依次填写四部分信息：

| 模块 | 操作 |
|------|------|
| **摄像头信息** | 拍摄像头标签照片 → 点击「识别文字」自动提取SN码 |
| **安装点位信息** | 拍点位环境照片 → 点击「获取当前定位」自动填入经纬度 |
| **光猫信息** | 拍光猫标签照片 → 自动识别SN码 → 填写宽带账号 |
| **机箱信息** | 拍机箱标签照片 → 点击「识别文字」提取编码 |
| **备注** | 填写其他说明 |

3. 点击 **「提交记录」** 保存

> 💡 **拍照技巧**：拍标签时保持手机平行于标签、光线充足，识别率更高

### 查看记录

- 首页列表显示所有记录，按时间倒序排列
- 点击任意记录进入详情页，可查看照片、定位地图
- 详情页右上角可 **编辑** 或 **删除**

### 搜索

- 列表页顶部搜索框，输入 SN 码 / 宽带账号 / 备注关键词
- 按回车或点搜索图标

### 导出数据

1. 点击底部 **「导出」**
2. 勾选需要导出的内容模块
3. 选择导出格式：
   - **Excel 报表**（.xlsx）— 含照片Base64数据
   - **CSV** — 纯文本格式
   - **JSON** — 原始数据（不含照片Base64）
4. 导出的文件会自动下载到手机

### 数据备份与恢复

- **导出备份**：导出含所有照片数据的完整备份文件（.backup.json）
- **恢复数据**：选择之前备份的文件，自动合并到当前设备
- 换手机或重装时，先导出备份，再在新设备上恢复

## 四、常见问题

### 打不开页面
- 确认电脑的服务器在运行（终端显示 `Server restarted`）
- 确认手机和电脑连的同一个 Wi-Fi
- 手机输入的是电脑IP，不是 localhost

### 拍照后没反应
- 首次使用会请求相机权限，需要允许
- iOS 端建议用 Safari，其他浏览器可能限制摄像头

### OCR 识别不出来
- 确保标签照片清晰、文字不倾斜
- OCR 需要下载语言包，首次使用耗时约 10-30 秒
- 可以手动输入SN码

### 定位获取失败
- 确保手机已开启 GPS 定位
- 在安全隐私中允许浏览器获取位置
- 室外定位更准确

### 数据会丢失吗
- 数据保存在手机本地浏览器中
- **清理浏览器缓存/数据会丢失所有记录**
- 建议定期导出备份（导出页 → 导出备份）

## 五、技术说明

- 纯前端应用，**不需要联网**也能使用（首次加载后）
- 所有照片和记录保存在手机本地
- 仅导出时需要网络连接生成文件
- 支持离线使用（PWA 缓存）
